(inner) fold(tf, seed, xs) → {any}

Works much like the Array::reduce function but accepts objects and all iterables which implement the @@iterator protocol
Parameters:
Name Type Description
tf function | transducer A transformer function or a transducer
seed any The seed value to transduce into
xs object | iterable The data to transduce over
Version:
  • 0.5.0
Source:
  • transducers.js, line 74
Returns:
The given data transduced into the seed value
Type
any
Example
const {fold} = require('futils').transducers;

const sep = (a, b) => !a ? b : a + ' - ' + b;
fold(sep, '', [1, 2, 3]); // -> '1 - 2 - 3'

const pair = (a, b) => a + ': ' + b;
fold(
    (acc, [val, key]) => sep(acc, pair(key, val)),
    '',
    {a: 1, b: 2, c: 3} 
);
// -> 'a: 1 - b: 2 - c: 3'
