    data. Id

    new Id()

    The Id data type. Id acts like a wrapper for any value to provide it with a bunch of interfaces
    Version:
    • 3.0.0
    Source:
    • data/Id.js, line 24
    Example
    const {Id} = require('futils').data;

Id(1); // -> Id(1)

Id(1).value; // -> 1
